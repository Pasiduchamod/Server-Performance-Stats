# Server-Performance-Stats

A simple Bash script to display system resource usage and basic OS information on Linux.

## Features
- Total CPU usage
- Free and used memory (with percentages)
- Disk usage (with percentages)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- OS information
- System uptime

## Requirements
- Linux OS
- `top`, `free`, `df`, `ps`, `awk`

## Usage
```bash
chmod +x system_monitor.sh
./system_monitor.sh
```
